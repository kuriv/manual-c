# 算术运算符

执行下面的代码，进行算术运算。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 233;
    int i2 = 666;
    printf("%d\n", i1 + i2);
    printf("%d\n", i1 - i2);
    printf("%d\n", i1 * i2);
    printf("%d\n", i1 / i2);
    printf("%d\n", i1 % i2);
    return 0;
}
```

整数除法会截断结果中的小数部分。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 10;
    int i2 = 3;
    printf("%d\n", i1 / i2);
    return 0;
}
```

执行下面的代码，判断是否是闰年。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int year = 2000;
    if ((year % 4 == 0 && year % 100 != 0) || year % 400 == 0)
    {
        printf("%d is a leap year.\n", year);
    }
    else
    {
        printf("%d is not a leap year.\n", year);
    }
    return 0;
}
```
