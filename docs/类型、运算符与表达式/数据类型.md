# 数据类型

C 语言提供了下列几种基本数据类型。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    char c = 'A';
    printf("%c\n", c);
    
    int i = 233;
    printf("%d\n", i);
    
    float f = 123.456;
    printf("%f\n", f);
    
    double d = 123.456;
    printf("%lf\n", d);
    return 0;
}
```

此外，还有 `signed` 与 `unsigned` 等类型限定符。
