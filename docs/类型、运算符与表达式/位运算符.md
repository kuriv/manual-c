# 位运算符

执行下面的代码，进行位运算。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 5;
    int i2 = 3;
    printf("%d\n", i1 & i2);
    printf("%d\n", i1 | i2);
    printf("%d\n", i1 ^ i2);
    printf("%d\n", i1 << 2);
    printf("%d\n", i1 >> 2);
    printf("%d\n", ~i1);
    return 0;
}
```
