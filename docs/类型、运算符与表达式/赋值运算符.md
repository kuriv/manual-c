# 赋值运算符

大多数二元运算符都有一个相应的赋值运算符。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 5;
    int i2 = 3;
    i1 += i2;
    printf("%d\n", i1);
    i1 -= i2;
    printf("%d\n", i1);
    i1 *= i2;
    printf("%d\n", i1);
    i1 /= i2;
    printf("%d\n", i1);
    i1 %= i2;
    printf("%d\n", i1);
    i1 <<= i2;
    printf("%d\n", i1);
    i1 >>= i2;
    printf("%d\n", i1);
    i1 &= i2;
    printf("%d\n", i1);
    i1 |= i2;
    printf("%d\n", i1);
    i1 ^= i2;
    printf("%d\n", i1);
    return 0;
}
```
