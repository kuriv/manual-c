# 指针与地址

指针是一种保存变量地址的变量。执行下面的代码，获取变量的地址。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
	int a = 233;
	int *p = &a;
	printf("%d\n", a);
	printf("%p\n", p);
    return 0;
}
```

执行下面的代码，通过指针获取并修改变量的值。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
	int a = 233;
	int *p = &a;
	printf("%d\n", *p);
	*p = 666;
	printf("%d\n", a);
	printf("%d\n", *p);
    return 0;
}
```
