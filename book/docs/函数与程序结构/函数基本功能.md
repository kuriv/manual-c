# 函数基本功能

执行下面的代码，定义并调用指定的函数。

```c
#include <stdio.h>

void print()
{
	printf("Hello World!");
}

int main(int argc, char const *argv[])
{
    print();
    return 0;
}
```

执行下面的代码，定义并调用实现两数加法的函数。

```c
#include <stdio.h>

int sum(int a, int b)
{
	return a + b;
}

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    printf("%d\n", sum(a, b));
    return 0;
}
```
