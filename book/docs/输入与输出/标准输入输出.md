# 标准输入输出

执行下面的代码，从标准输入中一次读取一个字符。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
	int c;
	c = getchar();
	printf("%c\n", c);
	return 0;
}
```

执行下面的代码，将输入的一个字符输出。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
	int c;
	c = getchar();
	putchar(c);
	return 0;
}
```

执行下面的代码，从标准输入中读取一行字符串。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
	int c;
	while ((c = getchar()) != EOF)
	{
		putchar(c);
	}
	return 0;
}
```
