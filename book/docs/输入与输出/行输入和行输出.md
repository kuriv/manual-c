# 行输入和行输出

执行下面的代码，从标准输入中读取一行，然后输出。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    char str[20];
    gets(str);
    puts(str);
    return 0;
}
```

执行下面的代码，从文件中读取一行，输出后再向文件中写入一行。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    FILE *fp;
    char str[20];
    fp = fopen("index.txt", "w+");
    fgets(str, 20, fp);
    puts(str);
    fputs("Hello C!", fp);
    fclose(fp);
    return 0;
}
```
