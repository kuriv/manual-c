# 行输入和行输出

执行下面的代码，从标准输入中读取一行，然后输出。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    char s[20];
    gets(s);
    puts(s);
    return 0;
}
```

执行下面的代码，从文件中读取一行，然后输出。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    FILE *fp;
    char s[20];
    fp = fopen("index.txt", "rw+");
    fgets(s, 20, fp);
    puts(s);
    fputs("Hello C!", fp);
    fclose(fp);
    return 0;
}
```
