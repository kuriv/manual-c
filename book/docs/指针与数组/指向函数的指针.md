# 指向函数的指针

函数本身不是变量，但可以定义指向函数的指针。

```c
#include <stdio.h>

int max(int x, int y)
{
    return x > y ? x : y;
}

int main(int argc, char const *argv[])
{
    int i1 = 233;
    int i2 = 666;
    int (*p)(int, int);
    p = max;
    printf("%d\n", max(i1, i2));
    printf("%d\n", p(i1, i2));
    return 0;
}
```
