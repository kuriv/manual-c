# 指向函数的指针

函数本身不是变量，但可以定义指向函数的指针。

```c
#include <stdio.h>

int max(int x, int y)
{
    return x > y ? x : y;
}

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    int (*p)(int, int);
    p = max;
    printf("%d\n", max(a, b));
    printf("%d\n", p(a, b));
    return 0;
}
```
