# 指针数组与指向指针的指针

由于指针本身也是变量，因此也可以像其他变量一样存储在数组中。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    char str1[] = "Hello World!";
    char str2[] = "Welcome!";
    char str3[] = "Hello C!";
    char *p, *q, *r;
    char *point[3];
    p = str1;
    q = str2;
    r = str3;
    point[0] = p;
    point[1] = q;
    point[2] = r;
    printf("%s\n", point[0]);
    printf("%s\n", *point);
    printf("%s\n", point[1]);
    printf("%s\n", *(point + 1));
    printf("%s\n", point[2]);
    printf("%s\n", *(point + 2));
    return 0;
}
```
