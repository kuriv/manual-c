# 位字段

在存储空间很宝贵的情况下，有可能需要将多个对象保存在一个机器字中。

```c
#include <stdio.h>

struct config
{
	unsigned int cfg1: 8;
	unsigned int cfg2: 16;
	unsigned int cfg3: 8;
};

int main(int argc, char const *argv[])
{
	struct config a;
	printf("size: %d.\n", sizeof(a));
	a.cfg1 = 255;
	a.cfg2 = 65535;
	a.cfg3 = 255;
	printf("cfg1: %d, cfg2: %d, cfg3: %d.\n", a.cfg1, a.cfg2, a.cfg3);
	return 0;
}
```
