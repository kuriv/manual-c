# 结构与函数

执行下面的代码，定义返回值为结构的函数。

```c
#include <stdio.h>

struct point
{
    int x;
    int y;
};

struct point initialize(int x, int y)
{
    struct point p = {x, y};
    return p;
}

int main(int argc, char const *argv[])
{
    struct point p = initialize(100, 100);
    printf("x: %d, y: %d.\n", p.x, p.y);
    return 0;
}
```
