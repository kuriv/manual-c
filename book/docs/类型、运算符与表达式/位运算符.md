# 位运算符

位运算符包括： & 、 | 、 ^ 、 << 、 >> 、 ~ 。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 5;
    int b = 3;
    printf("%d\n", a & b);
    printf("%d\n", a | b);
    printf("%d\n", a ^ b);
    printf("%d\n", a << 2);
    printf("%d\n", a >> 2);
    printf("%d\n", ~a);
    return 0;
}
```
