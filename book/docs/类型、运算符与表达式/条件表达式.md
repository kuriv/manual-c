# 条件表达式

执行下面的代码，进行条件判断。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 233;
    int i2 = 666;
    int max;
    if (i1 > i2)
    {
        max = i1;
    }
    else
    {
        max = i2;
    }
    printf("%d\n", max);
    return 0;
}
```

也可以使用三元运算符简化上面的代码。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 233;
    int i2 = 666;
    int max;
    max = i1 > i2 ? i1 : i2;
    printf("%d\n", max);
    return 0;
}
```
