# 条件表达式

执行下面的代码，进行条件判断。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    int z;
    if (a > b)
    {
        z = a;
    }
    else
    {
        z = b;
    }
    printf("%d\n", z);
    return 0;
}
```

也可以使用三元运算符简化上面的代码。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    int z;
    z = a > b ? a : b;
    printf("%d\n", z);
    return 0;
}
```
