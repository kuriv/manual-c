# 逻辑运算符

执行下面的代码，进行逻辑运算。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int i1 = 233;
    int i2 = 666;
    int i3 = 1;
    printf("%d\n", i1 > i2);
    printf("%d\n", i1 >= i2);
    printf("%d\n", i1 < i2);
    printf("%d\n", i1 <= i2);
    printf("%d\n", i1 == i2);
    printf("%d\n", i1 != i2);
    printf("%d\n", !i3);
    return 0;
}
```
