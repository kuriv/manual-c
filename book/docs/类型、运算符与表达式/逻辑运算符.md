# 逻辑运算符

逻辑运算符包括： > 、 ≥ 、 < 、 ≤ 、 == 、 != 、 ! 。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    int c = 1;
    printf("%d\n", a > b);
    printf("%d\n", a >= b);
    printf("%d\n", a < b);
    printf("%d\n", a <= b);
    printf("%d\n", a == b);
    printf("%d\n", a != b);
    printf("%d\n", !c);
    return 0;
}
```
