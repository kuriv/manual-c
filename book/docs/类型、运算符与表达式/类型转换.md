# 类型转换

当一个运算符的几个操作数类型不同时，就需要通过一些规则把它们转换为某种共同的类型。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 233;
    float b = 666.666;
    printf("%f\n", a + b);
    printf("%d\n", a + (int)b);
    return 0;
}
```

执行下面的代码，将字符串类型的数字转换为整型。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    char string[] = "123456";
    int a = 0;
    for (int i = 0; string[i] >= '0' && string[i] <= '9'; i++)
    {
        a = a * 10 + (string[i] - '0');
    }
    printf("%d\n", a);
    return 0;
}
```
