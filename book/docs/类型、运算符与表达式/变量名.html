变量名

变量名

C 语言变量名是由字母、数字和下划线组成的序列，但其第一个字符必须为字母，且严格区分大小写。

类似于 if 、 else 、 int 、 float 等关键字是保留给 C 语言本身使用的，不能用作变量名。
