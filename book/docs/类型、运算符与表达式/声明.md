# 声明

所有变量都必须先声明后使用。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 233;
    int b = 666;
    printf("%d\n", a);
    printf("%d\n", b);
    return 0;
}
```

一个声明指定一种变量类型，后面所带的变量表可以包含一个或多个该类型的变量。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a, b;
    a = 233;
    b = 666;
    printf("%d\n", a);
    printf("%d\n", b);
    return 0;
}
```

任何变量的声明都可以使用 `const` 限定符限定，该限定符指定变量的值不能被修改。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    const int a = 233;
    const int b = 666;
    printf("%d\n", a);
    printf("%d\n", b);
    return 0;
}
```

`const` 限定符也可配合数组使用，这表明数组所有元素的值都不能被修改。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    const char string[] = "I am a string.";
    printf("%s\n", string);
    return 0;
}
```
