# 赋值运算符

大多数二元运算符都有一个相应的赋值运算符。

```c
#include <stdio.h>

int main(int argc, char const *argv[])
{
    int a = 5;
    int b = 3;
    a += b;
    printf("%d\n", a);
    a -= b;
    printf("%d\n", a);
    a *= b;
    printf("%d\n", a);
    a /= b;
    printf("%d\n", a);
    a %= b;
    printf("%d\n", a);
    a <<= b;
    printf("%d\n", a);
    a >>= b;
    printf("%d\n", a);
    a &= b;
    printf("%d\n", a);
    a |= b;
    printf("%d\n", a);
    a ^= b;
    printf("%d\n", a);
    return 0;
}
```
